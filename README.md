## 차년도 발주를 위한 발주수량 예측

### 프로젝트 소개
선발주를 통해 재고를 관리하기 때문에 **차년도 발주 계획을 세우기 위한 수요 예측 모델을 만들었습니다.** 
과거 발주 데이터를 활용하여 발주 추이를 분석하였고, Seasonal ARIMA 모델을 활용하여 발주량 예측 모델을 구축했습니다.
